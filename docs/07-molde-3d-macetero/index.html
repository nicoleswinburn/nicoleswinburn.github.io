07 Molde 3D Macetero

Construcción Macetero Orgánico

En el siguiento post, daremos a conocer el proceso de modelado de un macetero orgánico. Primero explicaremos que materiales utilizaremos para la creación del macetero y luego el proceso del modelado del molde.

Materiales orgánicos del macetero

Para el macetero usaremos papel de cartón como base, con la técnica de papel maché. Como pegamento utilizaremos un engrudo de maicena con agua.

Materiales extras:
  • Café o té
  • Cáscara de huevo
  • Cáscara de plátano

Propiedades para las plantas:

Café:

El café es la bebida que se obtiene a partir de los granos tostados y molidos de los frutos de la planta del café. Los residuos del café se obtiene luego de que se se cole y prepare el café con un filtro en una cafetera. Los residuos del café tienen muchas propiedades saludables, aproximadamente un 1% del grano es lo que llega a la taza. Es así como el sobrante está cargados de nutrientes el cual se debe aprovechar.

Té:

Es un buen fertilizante. El té tiene ácidos y nitrógeno, que al añadirlos como infusión o en hojas de té negro se difunden en la base de la planta y ayudan a plantas como rosas y helechos a crecer.

Cáscara de huevo:

Es un fertilizante rico en calcio, también ayuda a mejorar el PH reduciendo la acidez del suelo.

Cáscara de plátano:

La cáscara de plátano o banana tiene múltiples beneficios para nuestras plantas. Su importancia estriba en las cantidades de potasio que contiene. Precisamente este mineral transporta los nutrientes por las plantas y fortalece la formación de flores y frutos.

Proceso modelado 3D

Hicimos el modelado en el programa Rhinoceros 3D. Para comenzar, el primer paso fue modelar el macetero. Para esto, creamos curvas con el diseño preestablecido y las medidas que queríamos que este tuviera. Las bases son de 80 mm x 80 mm, con una altura de 80 mm.

Luego aplicamos un loft a las curvas para crear la superficie.

Con el comando cap, rellenamos la figura para obtener un objeto sólido, al cuál le aplicamos un shell de 7 mm para convertirlo en un objeto sólido pero hueco.

Teniendo el macetero listo, procedimos a crear el molde de este. Para esto, dibujamos un cubo un poco más grande que el macetero para que lo contuviera dentro y realizamos un boolean difference entre estas dos piezas.

El resultado del paso anterior, son dos piezas (aparte del macetero): el molde exterior del macetero, que vendría siendo el molde negativo, y el contra-molde, que sería el molde positivo. Primero tomamos el molde exterior y lo separamos por la mitad con el comando split y le agregamos unos pequeños tabs redondos para que se mantenga firme cuando se esté formando el macetero.

Luego, tomamos la contraparte del molde, le agregamos dos cilindros en la parte inferior que ayudarán a crear dos pequeños orificios para que cuando la planta sea regada el agua pueda filtrar y no se ahogue la planta. Además, le creamos una manilla para poder facilitar el agarre y sustracción de dicha parte al momento de separar el molde y obtener el macetero.
